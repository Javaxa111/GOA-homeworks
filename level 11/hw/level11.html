Web Technologies

HTML

HTML is used to create the structure of a web page. It defines headings, paragraphs, images, links and more.

CSS

CSS is used to style the web page. It controls colors, fonts, layout and design.

JavaScript

JavaScript is used to add interactivity. For example: buttons, animations, form validation.

Difference

  • HTML – structure
  • CSS – design
  • JavaScript – behavior
